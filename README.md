```
caesar cipher implementation in x86 assembly language & rust
```
### Usage
- Clone repository
- Build:
    ```
    $ cd src/
    $ make
    ```
- Follow on-screen instructions
- Clean:
    ```
    $ make clean
    ```


### Rust
- Clone repository (git clone `git@github.com:gerardo-torres/caesar`)
- Build:
```
# cargo build --release
$ ./target/release/caesar <num> <text>
```
- Install:
```
$ cargo install
$ caesar <num> <text>
```
